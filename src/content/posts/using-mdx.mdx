This theme ships with [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) so you can author posts in MDX as well as Markdown.

## Why MDX?

MDX extends Markdown with JSX, so you can embed components where plain Markdown is not enough. See the [MDX in Astro](https://docs.astro.build/en/guides/integrations-guide/mdx/#mdx-in-astro) docs for patterns and limitations.

## Example

Interactive UI in MDX still follows Astro’s rules: use [client directives](https://docs.astro.build/en/reference/directives-reference/#client-directives) when you need hydration. For a simple link without a component:
